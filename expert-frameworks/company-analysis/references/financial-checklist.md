# 财务数据校验清单（勾稽关系核验）

> 本文件在输出任何财务表格前使用，确保数据一致性和逻辑正确性。

---

## 一、核心勾稽关系核验

在输出财务表格前，后台执行以下核验：

### 现金流量表内部核验

```
净利润（Net Income）
+ 折旧与摊销（D&A）
+ 股权激励（SBC）
+ 营运资本变化（Working Capital Changes）
± 其他非现金调整
= 经营现金流（CFO）                          ← 核验点1

经营现金流（CFO）
- 资本支出（Capex）
= 自由现金流（FCF）                          ← 核验点2

自由现金流（FCF）
- 股息（Dividends）
- 净回购（Net Buyback = Buyback - Stock Issuance）
± 并购净支出
= 净现金变化                                 ← 核验点3（与资产负债表现金变化对比）
```

### Cash ROE 计算核验

```
Cash ROE = FCF / 平均股东权益

平均股东权益 = (期初股东权益 + 期末股东权益) / 2

核验：期末股东权益 = 期初股东权益 + 净利润 - 股息 - 回购 + SBC + 其他
```

### FCF Yield 核验

```
FCF Yield = FCF / 市值（或 Enterprise Value）
注意：使用分析时点的市值，非年末市值
```

## 二、常见数据陷阱

| 陷阱 | 识别方法 | 处理方式 |
|-----|---------|---------|
| FCF被大额并购支出稀释 | 单独列出并购现金支出 | 计算有机FCF（剔除并购） |
| 高SBC导致FCF虚高 | CFO中SBC加回金额 | 注明SBC调整前后FCF |
| 营运资本波动导致CFO失真 | 检查应收/应付/存货变化 | 标注异常波动原因 |
| 维持性Capex vs 成长性Capex | 查管理层分类说明 | 使用维持性Capex计算真实FCF |
| 汇率影响 | 对比constant currency增速 | 注明汇率影响幅度 |

## 三、数据来源优先级

1. **最高优先**：公司官方SEC/交易所备案文件（10-K, 10-Q, 20-F, 港交所年报）
2. **次优先**：公司投资者关系官网发布的补充财务数据（Supplemental Data）
3. **参考**：Bloomberg/Refinitiv/FactSet（未来预测数据）
4. **谨慎使用**：财经媒体、第三方平台（需与原始报表交叉验证）

---

## 四、不确定性处理规则

| 情况 | 处理方式 |
|-----|---------|
| 找到数据但不确定是否最新 | 注明数据来源日期，提示可能已有更新 |
| 搜索不到某项数据 | 明确说明"未能获取"，不推测 |
| 数据与不同来源有出入 | 列出两个数据并说明差异原因 |
| 未来预测数据 | 标注"预测"，注明假设条件和来源 |

---

## 五、横向对比数据一致性

对比不同公司时，确保：
- 使用相同财年（有时不同公司财年结束时间不同）
- 使用相同计算口径（部分公司Capex定义不同）
- 汇率统一（建议使用美元，注明转换汇率）
- 如使用预测数据，来源和时间点需一致

---

*本清单在每次输出定量表格前执行内部核验，核验通过后方可输出结果。*
