## 一、三层分析框架：技术 × 产业 × 企业

### 1. 技术层分析

**a. 创新性评估**
- 独占性：是否有专利保护？Know-how壁垒多高？
- 创新速度：研发投入/收入比、专利申请趋势
- 技术路线判断：**原理领先 vs. 性能领先**（注意：真正的领先最终靠性能，不靠原理）

**b. 实现难度**
- 产品化难度（从实验室到可用产品）
- 产业化难度（从小批量到大规模量产）
- 用户接受难度（行为习惯改变的阻力）

**c. 产业影响类型**（三选一，决定竞争策略）
| 类型 | 定义 | 建议策略 |
|------|------|---------|
| 颠覆型（Disruptive） | 直接替代现有主流方案 | 低调进入，避免早期暴露 |
| 增强型（Enhancing） | 强化现有产业参与者能力 | 紧密与行业龙头合作，做Tier 1 |
| 新兴型（Nascent） | 创造全新市场/需求 | 大声讲故事，振臂一呼，教育市场 |

---

### 2. 产业层分析

**a. 产业生态框架（空间线）**

以目标公司为核心，绘制：
- 供方（直接供应商 → 供应商的供应商）
- 需方（直接客户 → 客户的客户）
- 竞争者（直接竞品 + 潜在替代）
- 合作者（生态伙伴、标准联盟）

识别**关键控制点**：谁掌握了标准/接口/数据？

**b. 产业影响框架（时间线）**

产业生命周期阶段：
```
新兴期 → 定型/稳定发展期 → 颠覆/升级期
```

判断当前所处阶段，识别"主导企业"（被别人影响小、影响别人大的核心企业）。

**c. 非连续性变化（Discontinuity）识别**

产业链重构信号：
- 技术路线切换（如从机械式到数字化）
- 价值链关键控制点转移
- 主导企业更迭

> 王煜全观点：产业链重构时是最大投资机会，要提前布局。

---

### 3. 企业层分析

**a. 企业发展阶段判断**

| 阶段 | 核心考察维度 | 关键指标 |
|------|------------|---------|
| 初创期（Start-up） | 技术优势、CEO决断力、融资能力 | 技术突破信号、核心团队、VC背书 |
| 成长期（Growth） | 管理能力、产品矩阵、高速扩张控制 | 收入增速、市场份额扩张、毛利趋势 |
| 成熟期（Maturity） | 生态化能力、并购布局、稳定回报 | FCF稳定性、战略投资质量、平台化程度 |
| 重构期（Restructuring） | 战略转型能力、业务剥离/重组 | 新业务占比、老业务衰退速度 |

**b. 成功信号识别**

| 信号类别 | 具体表现 |
|---------|---------|
| 技术信号 | 重大技术突破报道、顶级期刊论文 |
| 产品信号 | 产品原型面世、公测发布 |
| 量产信号 | 量产时间表确认、良率提升 |
| 市场信号 | 标杆客户签约、用户数快速增长 |
| 资本信号 | 顶级基金重仓、战略投资方进入 |

**c. 企业优势升级路径**

```
技术优势 → 产品优势 → 市场优势 → 产业优势 → 资本优势
（每一步升级都需要CEO能力的同步提升）
```

## 二、平台化分析框架

当公司有平台化迹象时，额外评估：

**平台成立的三个硬指标：**
1. 用户量够大（足以自我维持）
2. 利润够大（能养活足够多合作伙伴）
3. 足够阻止竞争（形成压制）

**平台生态制度设计评估：**
- 激励机制：是否有足够诱惑力让参与者留下？
- 准入机制：边界是否清晰，谁能进谁不能进？
- 惩罚机制：对违规行为是否有有效控制？

**平台型业务三个业务重点：**
- 粘性（Retention）
- 传染性（Virality）
- 冷启动能力（Cold Start）

## 三、技术进入市场的阶段模型

参考Carlota Perez框架 + 王煜全实践总结：

```
导入期（Installation Phase）
├── 阶段1：爆发 - 新技术在一个细分领域取得明显效果
├── 阶段2：狂热 - 资本大量涌入，估值泡沫形成
└── [泡沫破灭/金融危机]

展开期（Deployment Phase）
├── 阶段3：协同 - 技术广泛扩散，社会冲击蔓延
└── 阶段4：成熟 - 各行业普遍采用，社会进入良性增长
```

**投资意涵**：
- 导入期→展开期的临界点是最大的投资时机
- 泡沫是机会：企业应在泡沫期融更多资，填实业绩
- 展开期的"大水漫灌"会逐次抬升核心企业→同类企业→相关企业

## 四、科技企业估值特殊注意

王煜全核心观点：
> "科技企业有巨大拐点，传统财务指标会失效。用传统DCF分析蛰伏期的科技企业是错误的。"

**科技企业估值要点：**
1. 蛰伏期：看技术突破信号 + 市场潜力 + CEO能力，而非当期财务
2. 成长期：看收入增速 + 市场份额变化（市销率PSR比市盈率PE更重要）
3. 成熟期：看FCF稳定性 + Cash ROE + 生态化深度
4. 估值核心：市值管理的两个前提——**可预测、可预期、可量化**

**风险提示：**
- 新技术性能超越老技术需要时间（老技术经过长期调优，初期性能往往更优）
- 产业化时点判断是最难也最关键的环节
- 先发优势≠永久优势（技术出现转折时，后发可能反超）

## 五、Gartner曲线应用

```
技术萌芽 → 期望膨胀顶峰 → 泡沫破裂谷底 → 稳步爬升坡 → 生产成熟期
```

**投资策略对应：**
- 顶峰期：企业应融足够多的资，而非错过融资窗口
- 谷底期：真正的长期机会，但需要耐心
- 爬升期：趋势投资者进入，估值开始恢复
